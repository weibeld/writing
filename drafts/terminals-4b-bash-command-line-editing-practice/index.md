# Bash Command Line Editing (Practice)

Practical demonstrations of Bash/Readline line editing features and configuration.

## Content

- Useful Readline configurations
- Demonstrate difference between using `bind` and using a Readline init file
    - With `bind`, settings apply only to Bash, with Readline init file, settings apply to all apps that use Readline
- Practical command completion customisation (`complete`, `compgen`, `compopt`)
- History usage and customisation
- Switching between vi and emacs editing modes
